# Advanced-Sales-Insights-Dashboard-Interactive-Excel-Analytics-Dashboard
An interactive Excel Sales Dashboard built using Pivot Tables, Pivot Charts, Slicers, and Excel formulas to analyze sales performance across regions, products, customers, and months. Features dynamic KPIs, filters, and visualizations for data-driven business insights.
