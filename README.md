# 📊 Advanced Sales Insights Dashboard (Microsoft Excel)

An interactive **Sales Insights Dashboard** built in **Microsoft Excel** to analyze sales performance using Pivot Tables, Pivot Charts, Slicers, and Excel formulas. The dashboard provides dynamic KPIs and visualizations to help users monitor business performance and make data-driven decisions.

---

## 🚀 Project Overview

This project transforms raw sales data into an interactive dashboard that enables users to explore sales trends, customer behavior, regional performance, and product insights through dynamic filters and visual reports.

---

## ✨ Features

* 📈 Interactive Sales Dashboard
* 🎯 Dynamic KPIs

  * Total Sales
  * Total Orders
  * Total Quantity Sold
  * Average Order Value
* 🔍 Slicers for filtering by:

  * Region
  * Month
  * Product
* 📊 Pivot Tables & Pivot Charts
* 📅 Monthly Sales Analysis
* 🌍 Region-wise Performance
* 📦 Product-wise Sales Analysis
* 👥 Customer-wise Sales Insights
* 🏷️ Category-wise Analysis

---

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Excel Formulas
* Data Visualization

## 📌 How to Use

1. Download the Excel workbook.
2. Open it in Microsoft Excel (2019 or Microsoft 365 recommended).
3. Navigate to the **Dashboard** sheet.
4. Use the slicers to filter data by Region, Month, or Product.
5. Explore the automatically updated KPIs and charts.
